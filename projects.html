Console-Based Library Equipment Borrowing and Tracker System

A console-based system designed to help track borrowed school or library equipment, such as laptops, tablets, books, and other items. It allows the user to add items, record borrowers, monitor due dates, and log returns — all through a simple text-based interface. Built using clean programming logic suitable for beginners.
